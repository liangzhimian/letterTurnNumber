## 二十六进制 の 字母转数字

### 起源

    无意中看到一个题目
    A表示第1
    B表示2
    Z表示26
    AA表示27
    输入一个字符串，输出他表示第几

### 计算公式推导

    'ba' 表示 26 + 1
    两位数的时 'bc' ==> [2, 3]
    第2位数 3 代表 3个26 ==> 3 * 26^1  ==> 次方刚好是 2-1
    假设三位数 'cba' ==> [3, 2, 1]
    第3位数 1 代表 1个26² ==> 3 * 26^2 ==> 次方刚好是 3-1
    以此类推得到公式
    第i位数 n 代表 n个26的 i-1 次方 ==> n*26^(n-1)
    最后刚好26的零次方是1，满足需求
